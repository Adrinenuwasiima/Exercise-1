CSS SELECTORS

  • Element selector

    • This selects HTML elements based on the element name

  • Class selector

    • This selects HTML elements with a specific class attribute

  • ID selector

    • It is used to select one unique element

  • Descendant selector

    • It matches all elements that are descendants of a specified element

  • Direct descendant selector

HOW CSS SELECTORS ARE USED

  1. For the element selector: Write the element name followed by {}
  2. For the class selector: write a (.) character followed by the class name
  3. For the id selector: write a hash character followed by the id of the element
  4. For the descendant selector: Write (space) followed by the names of the descendant elements
  5. For direct descendant: use (>)
